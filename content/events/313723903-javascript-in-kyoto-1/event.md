Join us for our first JavaScript Hands-on in Kyoto.

Blow on the cartridge and press START. 🎮
In this workshop we’ll build our own web-based game controller and use it to battle in a retro 8-bit style tank game inspired by the classics. Instead of a joystick, you’ll create your own HTML controller and wire it up with your own event handlers.

This workshop is designed mostly for junior engineers, but anyone is welcome to join.

Player requirements:

- Laptop with git installed
- A working dev environment
- Ready to press START

By the end you’ll have buttons sending commands to a live game, some hands-on experience with WebSockets, and a chance to battle together.

Things we’ll use:

- React - the one and only
- HTML/CSS, of course we won't omit these two
- JavaScript
- WebSockets - this is where the magic happens

Depending on how fast we finish, we may try a few extra challenges or just hang out and do some networking. The workshop should take around 2 hours.
